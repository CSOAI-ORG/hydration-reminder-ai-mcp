# Hydration Reminder AI MCP

Track water intake, calculate hydration goals based on body weight, and send reminder schedules.

## Tools

- `log_water` — Log a water intake entry in ml or oz.
- `calculate_daily_goal` — Calculate recommended daily water intake.
- `get_hydration_status` — Return progress toward daily goal and reminders.

## Install

```bash
pip install -e .
```

## Run

```bash
python -m hydration-reminder-ai-mcp.server
```
